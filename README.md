# Content Sharing Platform 🚀

A full-stack web application designed for real-time content sharing with persistent data storage.
<img width="1920" height="1080" alt="Screenshot (28)" src="https://github.com/user-attachments/assets/dbdfec4a-4510-4da3-ab06-74357da3a432" />


### 🛠 Tech Stack
* **Language:** Python
* **Web Framework:** Flask
* **Database:** SQLite (Relational Database)
* **Design:** HTML5 & Tailwind CSS

### 💡 Key Technical Achievement
* Transitioned from volatile local memory to **Relational Database Management (SQLite)**, ensuring data persistence and integrity.
* Implemented a clean **CRUD** (Create, Read, Update, Delete) architecture for user-generated content.

### 📋 How to run
1. Clone the repository: `git clone [URL]`
2. Install dependencies: `pip install flask`
3. Run the application: `python app.py`
